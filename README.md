
# Technical Communication Portfolio (UCF Minimal)

Forkable GitHub Pages template with a simple black/gold navbar and minimal placeholder text.

## Instructor setup
1. Create a public repo and upload these files.
2. Mark the repo as a **Template repository**.
3. Enable **Pages**: Deploy from branch → `main` (root).

## Student usage
- Edit the assignment pages under `/_assignments/`.
- Upload `assets/docs/resume.pdf` and keep the link on the Resume page.
- Publish via **Settings → Pages**.

## Colors
Black `#000000`, Bright Gold `#FFC904` (digital).
